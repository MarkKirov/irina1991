# Project Memory

## Core
- App language and content must be strictly in Russian.
- Supabase (Edge Functions, DB) for backend. App is distributed as a PWA.
- Minimalist, feminine design (sage, dusty rose, beige). Orange for CTA.
- All session data currently saved to `localStorage` for auto-restore.
- Lead-magnet productivity app for women to sell coaching mentorship.
