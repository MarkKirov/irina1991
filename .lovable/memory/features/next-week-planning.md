---
name: Next week planning
description: Переключатель недель на дашборде для планирования задач на следующую неделю
type: feature
---
На дашборде добавлен переключатель «Текущая неделя» / «Следующая неделя».
- Задачи имеют поле `week: number | null` для привязки к конкретной неделе.
- При просмотре следующей недели скрываются блоки месяц/ежедневно/привычка и кнопки PDF/AI/отчёт.
- Можно добавлять задачи на следующую неделю и назначать их на дни.
- При вызове `startNextWeek` задачи следующей недели становятся текущими.
